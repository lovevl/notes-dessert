完整的indexof用法：indexOf（str，fromIndex）表示从索引位置fromIndex开始查找，如果fromIndex省略，则表示默认从起始索引0开始查找； 若fromIndex为负，则从索引0开始查找。返回的索引值仍然是在整个字符串中的索引值

数组容器

shift unshift 从数组开头开始添加或者删除 pop push 从数组末尾添加或者删除
shift：删除原数组第一项，并返回删除元素的值；如果数组为空则返回undefined

var arr1 = [1,2,3,4,5];

var result = arr1.shift(); //arr1=[2,3,4,5] result=1

unshift：将参数添加到原数组开头，并返回数组的长度

result = arr1.unshift(-2,-1); //arr1=[-2,-1,1,2,3,4,5] result=7

pop：删除原数组最后一项，并返回删除元素的值；如果数组为空则返回undefined

result = arr1.pop(); //arr1=[1,2,3,4] result=5

push：将参数添加到原数组末尾，并返回数组的长度

result = arr1.push(6,7); //arr1=[1,2,3,4,5,6,7] result=7

setSeconds(sec, opt_msec) ：设置Date对象的秒数值。可选毫秒

正则表达式基础知识

星号(*) : 星号代表匹配它前面一个字符任意遍(0或任意次)

加号(+): 加号是一个与星号(*)类似的通配符，它也是数量词，表示匹配前面的字符一次或多次（至少一次).

问号(?): 问号也是一个数量词，它代表匹配前一个字符0或1次。

中括号[]： 中括号用来表示一个字符集合，如:[a-z]表示小写字母的集合，

脱字符^ (caret).表示，匹配任何不在该集合中的字符，与上面的用法刚好相反

\w -- (小写w) 表示字母或数字，等价于 [a-zA-Z0-9]

\W -- (大写W)非字母且非数字，与\w相反 等价于 '[^A-Za-z0-9_]'

\s -- (小写s)匹配一个空格字符，包括：空格，换行，回车,tab,等价于[ \n\r\t\f]

\S -- (大写S)匹配非空格字符，\s的相反 等价于 [^ \f\n\r\t\v]。

\d -- 表示10进制数字，等价于 [0-9]

\D -- 匹配一个非数字字符。等价于 [^0-9]。

\f 匹配一个换页符。等价于 \x0c 和 \cL。

\n 匹配一个换行符。等价于 \x0a 和 \cJ。

\r 匹配一个回车符。等价于 \x0d 和 \cM。

\t 匹配一个制表符。等价于 \x09 和 \cI。

\v 匹配一个垂直制表符。等价于 \x0b 和 \cK。

大括号:{} 指定重复前面一个字符多少遍:{N} 重复N遍;{n,m} 重复 n~m 遍;{n,} 至少重复n遍;{,m} 至多重复m遍

^ 匹配输入字符串的开始位置。

$ 匹配输入字符串的结束位置。

\b 匹配一个单词边界，也就是指单词和空格间的位置。//例如， 'er\b' 可以匹配"never" 中的 'er'，但不能匹配 "verb" 中的 'er'。

\B 和\b相反，匹配非单词边界。'er\B' 能匹配 "verb" 中的 'er'，但不能匹配 "never" 中的 'er'。

replace用法

基础用法：如果要替换全部匹配项，需要传入一个 RegExp 对象并指定其 global 属性。

myString = "javascript is a good script language";

console.log(myString.replace("a","A")); //只替换了第一个字符a

console.log(myString.replace(/a/,"A")); //只替换了第一个字符a

console.log(myString.replace(new RegExp('a','gm'),"A")); //全部替换

console.log(myString.replace(/a/g,"A")); //全部替换

高级用法：特殊标记$

对于正则replace约定了一个特殊标记符$：

1.$i (i:1-99) : 表示从左到右正则子表达式所匹配的第i组的文本。

案例1 - 匹配后替换:将把所有的花引号替换为直引号：

myString = '"a", "b"';

myString = myString.replace(/"([^"]*)"/g, "'$1'"); //'a', 'b'

案例2 - 匹配后替换

myString= "javascript is a good script language";

console.log(myString.replace(/(javascript)\s*(is)/g,"$1 $2 fun. it $2")); //javascript is fun. it is a good script language

案例3 - 分组匹配后颠倒

myString = "itcast , cn";

myString = myString.replace(/(\w+)\s*,\s*(\w+)/, "$2 $1"); //cn itcast

案例4 - 分组匹配后颠倒

myString = "boy & girl";

myString.replace(/(\w+)\s*&\s*(\w+)/,"$2 & $1"); //理论上：girl & boy但结果没变

myString = "boy + girl";

myString.replace(/(\w+)\s*\+\s*(\w+)/g,"$2 $1"); //理论上：girl boy但结果没变

myString.replace(/(\w+)\s*\+\s*(\w+)/g,function(m,p1,p2){

   console.log(p2+" & "+p1);

}); //girl & boy 该方法参考以下 高级用法

2.$&:表示与正则表达式匹配的全文本。

myString = "boy";

myString.replace(/\w+/g,"$&-$&") //理论上：boy-boy但结果没变

3.$`(`:切换技能键)：表示匹配字符串的左边文本。

myString = "javascript";

myString.replace(/script/,"$& != $`") //理论上：javascript != java 但结果没变

4.$'(‘:单引号)：表示匹配字符串的右边文本。

myString = "javascript";

myString.replace(/java/,"$& is $'") //理论上： javascript is script 但结果没变

5.$$：表示$转移。
高级用法：第二个参数可以是函数

myString = "bbabc";

myString.replace(/(a)(b)/g, function(){

  console.log(arguments) // ["ab", "a", "b", 2, "bbabc"]

});

参数将依次为：

1、整个正则表达式匹配的字符。

2、第一分组匹配的内容、第二分组匹配的内容…… 以此类推直到最后一个分组。

3、此次匹配在源自符串中的下标（位置）。

4、源自符串

举例1 首字母大写 -- 一个参数 表示匹配的整个字符串

myString = 'aaa bbb ccc';

myString=myString.replace(/\b\w+\b/g, function(word){

  return word.substring(0,1).toUpperCase()+word.substring(1);

}); // Aaa Bbb Ccc

举例2 首字母大写 -- 多个参数 - 第一个表示匹配的整个字符串，后面的表示分组中的内容

myString = "i am a boy !"

myString = myString.replace( /(^|\s)([a-z])/g , function(m,p1,p2){

  return p1+p2.toUpperCase();

}); //I Am A Boy！

//当同一块区域内有多个触发事件（例如，多个div互相嵌套，各有各的click事件），

//用于防止这多个触发事件都发生响应。目的只需响应被触发的那个事件

    btn.onclick = function(event) {

        alert("点击了按钮")

        var event = event || window.event;

        if(event && event.stopPropagation)

        {

            event.stopPropagation();  //  w3c 标准

        }

        else

        {

            event.cancelBubble = true;  // ie 678  ie浏览器

        }

    }

阻止默认行为

例如a标签的href属性或form标签的onsubmit属性，有自身的行为，当我们需要定义自己的事件时，该方法可以阻止自身行为触发

    a.on("click",function(event){

        var event = event?event:window.event;

        if(event.preventDefault){

            event.preventDefault();

        }else{

            event.returnValue = false;

        }

    });
